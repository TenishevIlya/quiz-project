# Quiz-project
Квиз система c возможностью администрирования и публичного использования

# Деплой

### Run project
`npm run server`

## Run json-server
`npm run json-server`

### Compiling js and scss
`npm run build:js`

`npm run build:css`

### Search js and css changes
`npm run dev:js`

`npm run dev:css`

## Run linter
`npx eslint filename.js`
