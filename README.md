# Quiz-project
Квиз система c возможностью администрирования и публичного использования

# Деплой

#### Инициализация проекта
` npm init`

### Установка lite-server
`npm install lite-server --save-dev`

### Запуск проекта
`npm start`

### Компиляция js и scss
`npm run build:js`

`npm run build:css`

### Отслеживание изменений js и css
`npm run dev:js`

`npm run dev:css`
